# Vegetable sale

Weekly sale of frozen vegetables in different stores.

## Usage

``` r
vegetables
```

## Format

A data frame with 1056 rows and 3 columns:

- `sale`:

  a numeric vector. Number of items sold in a week

- `normalSale`:

  a numeric vector. Estimated normal sale that week

- `store`:

  a factor. Id number of the different stores
