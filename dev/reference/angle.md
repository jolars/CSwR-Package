# Torsion angles

The phi and psi torsion angles between peptide planes for the human
protein 1HMP.

## Usage

``` r
angle
```

## Format

A data frame with 419 rows and 5 columns:

- `chain`:

  a character. The chain (either A or B) of the protein

- `AA`:

  a character. The three-letter amino acid name

- `pos`:

  an integer. Position in the amino acid sequence

- `phi`:

  numeric. The phi angle

- `psi`:

  numeric. The psi angle
